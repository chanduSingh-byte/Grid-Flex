Flexbox Container

Definition: Flexbox (display:flex) is a one-dimensional layout mode. It arranges children along a single line called the main axis, and lets you align them across the perpendicular cross axis. It shines whenever content needs to flow, wrap, or share leftover space — not when you need a fixed two-dimensional grid.

Grid Container

  

  
Definition: Grid (display:grid) is a two-dimensional layout mode. It carves the container into explicit rows and columns up front — a track structure — then places children into that structure by coordinate, not by document flow order. Use it when the layout itself needs structure, not just a sequence of items.

Child / Item Properties

  

  
Definition: these properties are set on the children, not the container. They override how one specific item behaves inside the layout its parent already established — so a container decides the rules, and an item can break one of those rules for itself. Every rig below is live: click a chip to change that item's property.


  


    
    

      
flex-grow / flex-shrink / flex-basis

      
grow = how much of the leftover free space an item claims, relative to siblings. shrink = how much it's allowed to shrink below its basis when space runs out. basis = its starting size before grow/shrink are applied.
