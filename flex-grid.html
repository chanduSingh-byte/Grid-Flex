    03
    
Child / Item Properties

  

  
These apply to the children, not the container — they override how one item behaves inside the layout its parent already set up.


  


    
    

      
flex-grow / flex-shrink / flex-basis

      
Controls how a flex item stretches, shrinks, or starts sized before free space is distributed.

      

        
grow:1

        
grow:2

        
grow:1

      

      
.item { flex-grow: 2; }
.item { flex-shrink: 0; }
.item { flex-basis: 200px; }

/* tailwind */
class="grow-[2]"
class="shrink-0"
class="basis-[200px]"

    


    
    

      
order

      
Reorders items visually without touching the HTML source order. Lower numbers render first.

      

        
A · order:3

        
B · order:1

        
C · order:2

      

      
.a { order: 3; }
.b { order: 1; }
.c { order: 2; }

/* tailwind */
class="order-3"
class="order-1"
class="order-2"

    


    
    

      
align-self

      
Overrides the container's align-items for just one item — works in both flex and grid.

      

        
start

        
center

        
end

      

      
.item { align-self: center; }

/* tailwind */
class="self-center"
class="self-start"
class="self-end"

    


    
    

      
grid-column / grid-row span

      
Lets one grid child stretch across multiple tracks — the classic "featured card" pattern.

      

        
span 2 cols

        
1×1

        
1×1

      

      
.feature {
  grid-column: span 2 / span 2;
  grid-row: span 1;
}

/* tailwind */
class="col-span-2 row-span-1"
